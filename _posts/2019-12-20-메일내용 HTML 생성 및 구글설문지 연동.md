# 메일 내용 HTML 생성
  
이미지 버튼 클릭하면 해당 링크로 이동하는 HTML 코드이다!  
동의 이미지 버튼 클릭-> 구글폼 객관식 문항에서 동의를 value로 갖는 라디오 버튼 값이 클릭됨  
거절 이미지 버튼 클릭-> 구글폼 거절 응답 자동 제출. 추가 제출 클릭 필요 없음  

html 코드는 이렇게 작성함...    
수신자는 아래처럼 나타나는 메일을 받을 수 있게 된당!
